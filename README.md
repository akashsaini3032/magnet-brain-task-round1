Task Management System (MERN Stack)

 Note: The .env file is intentionally included in .gitignore because it contains sensitive information such as the email ID and password used with Nodemailer to send emails to users.




 Project Overview

This is a Simple Task Management System built with the MERN stack (MongoDB, Express.js, React, Node.js).
It allows users to create, edit, delete, and manage tasks with authentication and email notifications.


Features
1.  Task Creation

Users can create new tasks with a title, description, due date, and assign them to priority lists.

2.  Task List

Displays all tasks with pagination and Ajax support.

Shows title, due date, and status (pending, completed).

3.  Task Details

View details of a specific task including its description and due date.

4.  Task Editing

Edit existing tasks (update title, description, and due date).

5.  Task Deletion

Delete a task with a confirmation dialog before final deletion.

6.  Task Status Update

Mark tasks as completed or change their status dynamically.

7.  User Authentication

Secure login and registration using JWT and bcrypt.

Only authorized users can create, view, edit, or delete tasks.

8.  Priority Management

Move tasks between different priority lists (High, Medium, Low).
